* J1939 hacking
** 综述
- J1939是我的一个知识短板，在过去多年的工作中算是一个不小的遗憾。在过去的这么多年中，我似乎就没见过真正的J1939运行起来应该是什么样子。这次简单的学习，用了一些开源的代码以及工具，就当他们的运行效果是真的J1939的味道了。
- 我找的教程基本上以微芯的开源例程为原本，为了保证软件的原汁原味，我还买了两个微芯的两个开发板。
